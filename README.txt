# Capston Project

The project is : 
The player tries to find a truck toy in the store and gives it to the kid for him to stop crying and become happy during 3 minutes.

Emotions: Crying & Laughing.

## Versions
- Unity 2017.4.15
- SteamVR v2.3.2
##
